EXERCÍCIO 1 – CLASSE DATA
UFCD 10794 – Programação Avançada com Python

Implementação de uma classe Data com:
- Validação de datas (incluindo anos bissextos)
- Formatação no formato DD/MM/AAAA
- Avanço da data em 1 dia

Ficheiros:
- ex1_classe_data.py  -> implementação da classe
- test_data.py       -> testes unitários com unittest

Como executar os testes:
python -m unittest test_data.py
