EXERCÍCIO 2 – CLASSE ALUNOPROVAS

Implementação de uma classe AlunoProvas com:
- Registo de provas (3 notas por prova)
- Validação de notas entre 0 e 20
- Cálculo da média geral
- Atribuição de conceito final (A, B, C, R)

Ficheiros:
- ex2_classe_aluno_provas.py  -> implementação da classe
- test_aluno_provas.py       -> testes unitários com unittest

Como executar os testes do exercício 2:
python -m unittest test_aluno_provas.py
